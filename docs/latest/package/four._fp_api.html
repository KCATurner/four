Convert a number to a list of (P, R) tuples.

Parameters:

number (int | float) – A number to convert.

Returns:

A list of (P, R) tuples representing the given number, where P is a period value to be repeated R number of times before moving on to the next tuple in the list.

Return type:

List[Tuple[int, int]]

Examples

>>> number_to_periods(123000000000)
[(123, 1), (0, 3)]

>>> number_to_periods(1.23e17)
[(123, 1), (0, 3)]

four._fp_api.parse_abbreviation(abbreviation: str) → List[Tuple[int, int]][source]

Retrieve a number from its given string abbreviation.

Parameters:

abbreviation (str) – A pseudo-numeric string in which repeated periods are represented like “[P]{R}” where P is the period value and R is the number of times the period is repeated.

Returns:

A list of (P, R) tuples representing a number, where P is a period value to be repeated R number of times before moving on to the next tuple in the list.

Return type:

List[Tuple[int, int]]

Examples

>>> parse_abbreviation("123[456]{2}")
[(123, 1), (456, 2)]

>>> parse_abbreviation("[987]{6}543543")
[(987, 6), (543, 2)]
