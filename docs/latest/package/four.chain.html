four.chain

Calculate and print 4-chains of specified target lengths from various root nodes.

CLI Subcommand

four chain

print four-chains of target lengths

usage: four chain [-h] [-v] [-l LENGTH] [-c CHAIN_INDEX]
-h, --help

show this help message and exit

-v, --verbose

increase output verbosity

-l <length>, --length <length>

length of the target chain; default: 1

-c <chain_index>, --chain <chain_index>

the target chain (by 1-based index)
