four.graph

Generate 4-chain tree graphs of various sizes given varying root nodes.

CLI Subcommand

four graph

generate graphs of four-chain trees

usage: four graph [-h] [-y] [-d] [-o OUTPUT] [-r ROOT] [-l LIMIT] [-e ENGINE]
-h, --help

show this help message and exit

-y, --yes

skip warnings with y/yes

-d, --directional

make graph edges directional

-o <output>, --output <output>

save image to output file

-r <root>, --root <root>

set the root node of the graph; default: 4

-l <limit>, --limit <limit>

maximum number to include in the tree; default: 50

-e {circo,dot,fdp,neato,osage,patchwork,sfdp,twopi}, --engine {circo,dot,fdp,neato,osage,patchwork,sfdp,twopi}

change the layout of the graph nodes; choices: circo, dot, fdp, neato, osage, patchwork, sfdp, twopi default: dot
