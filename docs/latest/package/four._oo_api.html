class four._oo_api.RPeriod(value: int | Tuple[int, int] | RPeriod, repeat: int = 1)[source]

Basic container for a repeated period.

One or more consecutive periods in a number with the same value represented as a tuple-like container (P, R) where P is a valid period value in the range [0, 1000), and R is the number of times P is repeated.

Parameters:

  • value (int | Tuple[int, int] | RPeriod) – The value of the period.

  • repeat (int) – Number of times this period repeats.

Examples

>>> RPeriod(123)
(123, 1)

>>> RPeriod(123)
(12, 3)

>>> RPeriod((1, 23))
(1, 23)

class four._oo_api.PNumber(number: int | str | Iterable[int | Tuple[int, int] | RPeriod] | None = None, debug: bool = False)[source]

A number as a list of RPeriods.

A list of RPeriods like (P, R) where P is the value of the RPeriod and R is the number of times that period repeats.

Parameters:

  • number (int | str | Iterable[int | Tuple[int, int] | RPeriod]) – A number or sequence of RPeriods.

  • debug (bool) – If true, print debug information whenever certain attributes are calculated. Generally only inteneded for use with CLI

Examples

>>> PNumber(123000000000)
[(123, 1), (0, 3)]

>>> PNumber("123[000]{3}")
[(123, 1), (0, 3)]

>>> PNumber([(123, 1), (0, 3)])
[(123, 1), (0, 3)]
